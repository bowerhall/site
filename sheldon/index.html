sheldon

An assistant that remembers what matters to you, learns your rhythms, and develops personality over time. Whatever you need done on a computer, Sheldon can probably do it. Self-hosted for ~$8/mo.

Install on any Ubuntu VPS
curl -fsSL bowerhall.ai/sheldon/install | sudo bash
Memory
  • 14 life domains Identity, health, relationships, work, finances, goals, routines. Facts tagged and retrievable by domain.
  • Contradiction resolution "I moved to Portland" supersedes "I live in NYC". Old facts archived, new facts active.
  • Time-based recall "What did we discuss last week?" "What happened on February 20th?" Explicit time queries, not just decay.
  • Zero-cost embeddings Local Ollama models. No API fees for vector search.
Crons that remember
  • Memory-linked triggers Crons store a keyword, not a message. At fire time, Sheldon recalls current context matching that keyword.
  • Adapts to changes "Take meds at 8pm" → later you mention "take with food" → the 8pm reminder now includes both facts.
  • One-time or recurring "Remind me at 3pm" fires once. "Check on me every 6 hours" runs until — you decide.
Code to deploy
  • Sandboxed coder Ephemeral Docker containers. No access to your tokens. Writes code, can't push directly.
  • GitHub collaboration Add an org-scoped PAT. Sheldon creates branches and opens PRs—no access to main. You review and merge.
  • Full pipeline Request → code → Docker build → registry push → Traefik routes → live at https://app.yourdomain.com
  • Multi-provider Kimi, Claude, GPT-4, Ollama. Ask Sheldon to switch models mid-conversation. Auto-fallback when credits run out.
  • Local models Ask Sheldon to pull and run Ollama models. He'll check disk space before downloading.
Vision
  • Images and videos Send a receipt, screenshot, or video. Sheldon extracts the info and remembers it.
  • Save and recall Images saved to MinIO. Ask "what was on that receipt?" and Sheldon knows.
  • Model-dependent Works with Claude, GPT-4o, and vision-capable Ollama models. Sheldon tells you if the current model can't see.
  • Vision + notes + crons Photo of pill bottle → extracts dosage → 8pm cron reminds with correct dose. Screenshot of gym plan → saves to note → weekly check-in tracks progress. Photo of whiteboard → extracts tasks → Sheldon follows up on each.
  • ADHD-friendly Crons recall your context and know what changed before nudging you. Stay organized without actually organizing. Just share things with Sheldon and he handles the rest.
Browser
  • Browser automation Sandboxed agent-browser for JS-heavy sites. Research, form filling, scraping.
  • Apartment hunting Browse listings, compare options, fill applications. Works in foreign languages too.
  • Job applications Apply to jobs with custom resumes. Sheldon tailors each application to the posting.
  • Extensible with skills Teach Sheldon new workflows via markdown skills. If it can be done in a browser, Sheldon can learn it.
Homelab
  • Private mesh network Headscale (self-hosted Tailscale). Add machines with one command. Manage containers remotely.
  • S3 storage MinIO for files, backups, sharing. Generate expiring links. Backup memory to storage.
  • No config files Ask Sheldon for disk space, container health, or logs. Restart your services, clean up. He can't touch his own core infra.
  • Free tier forever Switch to local models and pay nothing beyond your $8/mo VPS.
