sheldon

An assistant that remembers what matters to you, learns your rhythms, and develops personality over time. Whatever you need done on a computer, Sheldon can probably do it. Self-hosted for ~$8/mo.

Install on any Ubuntu VPS
curl -fsSL bowerhall.ai/sheldon/install | sudo bash
Memory
  • 14 life domains Identity, health, relationships, work, finances, goals, routines. Facts tagged and retrievable by domain.
  • Contradiction resolution "I moved to Portland" supersedes "I live in NYC". Old facts archived, new facts active.
  • Time-based recall "What did we discuss last week?" "What happened on February 20th?" Explicit time queries, not just decay.
  • Zero-cost embeddings Local Ollama models. No API fees for vector search.
Crons that remember
  • Memory-linked triggers Crons store a keyword, not a message. At fire time, Sheldon recalls current context matching that keyword.
  • Adapts to changes "Take meds at 8pm" → later you mention "take with food" → the 8pm reminder now includes both facts.
  • One-time or recurring "Remind me at 3pm" auto-deletes after firing. "Check on me every 6 hours" persists.
Code to deploy
  • Sandboxed coder Ephemeral Docker containers. No access to your tokens. Writes code, can't push directly.
  • GitHub collaboration Creates branches, opens PRs to your org. You review and merge.
  • Full pipeline Request → code → Docker build → registry push → Traefik routes → live at https://app.yourdomain.com
  • Multi-provider Kimi, Claude, GPT-4, Ollama. Switch models mid-conversation. Auto-fallback when credits run out.
Browser
  • Browser automation Sandboxed agent-browser for JS-heavy sites. Research, form filling, scraping.
Homelab
  • Private mesh network Headscale (self-hosted Tailscale). Add machines with one command. Manage containers remotely.
  • S3 storage MinIO for files, backups, sharing. Generate expiring links. Backup memory to storage.
