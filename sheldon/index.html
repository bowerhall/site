Sheldon Live

Personal AI assistant that remembers your entire life, runs on your infrastructure, and writes code autonomously.

Install on any Ubuntu VPS:

curl -fsSL bowerhall.ai/sheldon/install | sudo bash
  • Structured memory — 14 life domains, semantic search, confidence-scored facts
  • Autonomous coder — Writes, tests, and deploys code in sandboxed containers
  • Web browsing — Researches, fills forms, handles JS-heavy sites
  • Context-aware crons — Reminders and check-ins that understand context
  • Multi-provider LLM — Kimi, Claude, OpenAI, Ollama. Switch at runtime
  • Self-hosted — Your data stays on your infrastructure
