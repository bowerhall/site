sheldonmem Live

Embeddable memory system for AI agents. Single SQLite file, zero external dependencies.

Install:

go get github.com/bowerhall/sheldon/pkg/sheldonmem
  • Single file storage — SQLite + sqlite-vec, portable, no external DB
  • 14 life domains — Structured memory across identity, health, work, relationships, etc.
  • Entity-fact-edge model — Knowledge graph with typed relationships
  • Hybrid retrieval — Keyword + semantic search combined
  • Confidence scoring — Facts have confidence levels, contradictions auto-resolve
  • Memory decay — Salience-aware cleanup for long-running agents

Quick start:

store, _ := sheldonmem.Open("/path/to/memory.db") store.SetEmbedder(yourEmbedder) result, _ := store.Recall(ctx, "user's location", nil, 5)
