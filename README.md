# bowerhall.ai

Landing page for [Bowerhall](https://bowerhall.ai) - opensource applied AI infrastructure.

## Projects

- **sheldon** - AI assistant that remembers what matters
- **sheldonmem** - Knowledge graph memory for AI agents
- **gasket** - Redaction layer between your data and your agent

## Development

Static HTML/CSS. Open `index.html` in a browser or serve locally:

```bash
npx serve .
```

## Deployment

Deployed via Cloudflare Pages. Pushes to `main` auto-deploy.
